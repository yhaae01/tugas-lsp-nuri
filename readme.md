## Spesifikasi Minimum

- PHP >= 5.6 atau terbaru.
- Google Chrome >= 89.0.\*

## Cara Menggunakan

1. Simpan folder Pustaka ke dalam
   > C:\xampp\htdocs\*
2. Jalankan web Server Apache dan MySQL
3. Import database dari folder database
4. Buka web browser lalu ketikan localhost/pustaka
